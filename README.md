# Vocabulary Learning Web App

## Overview

The Vocabulary Learning Web App is a lightweight, browser-based application designed to help users learn and manage vocabulary effectively. The app allows users to add, edit, delete, and search for vocabulary words, all while ensuring that their data is saved locally in the browser. It is built using modern web technologies and is optimized for mobile devices.

## Features

- **Vocabulary Word Management**: Users can add, edit, delete, and search for vocabulary words.
- **Local Storage Persistence**: All user data is saved in the browser using localStorage, ensuring that data is retained even after refreshing the page.
- **Offline Functionality**: The app works fully offline after the initial load.
- **Mobile-Friendly Design**: The app is designed to be clean and user-friendly on mobile devices.

## Setup Instructions

1. **Clone the Repository**: Clone this repository to your local machine using `git clone <repository-url>`.
2. **Open the App**: Open `index.html` in your web browser to start using the app.
3. **Usage**: Follow the on-screen instructions to manage your vocabulary words.

## Usage Guidelines

- To add a new vocabulary word, fill in the required fields and click the "Add" button.
- To edit an existing word, click on the word in the list, make your changes, and save.
- To delete a word, click the delete button next to the word you wish to remove.
- Use the search bar to filter words based on your input.
